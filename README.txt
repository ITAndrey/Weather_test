
Проект: Запуск с использованием Docker-compose

Этот проект использует Docker-compose для облегчения развертывания и запуска. В данном README содержится инструкция по созданию и запуску контейнера.

Предварительные требования

Перед началом убедитесь, что у вас установлены:
- Docker (https://www.docker.com/get-started)
- Docker Compose (https://docs.docker.com/compose/install/)

Использование Docker Compose 

В терминале переходим в папку Testproject, тут нужно прописать следующую команду

docker-compose up

После того как все системы запустятся и пройдут тесты в адресной строке браузера введите http://0.0.0.0:8000 
На этом сайте вы можете проверить температуру любого города, а также посмотреть историю запросов. Когда вы перейдете в историю запросов, у вас будет возможность удалить любой запрос, а также настроить фильтрацию под себя (этот функционал можно очень сильно расширить в дальнейшем)

Также вы можете перейти в Telegram bot и узнать всю эту информацию в нем ссылка на бота https://t.me/SuperWeathererBot.
Данный бот обладает таким же функционалом как и сайт.

Для остановки docker-compose нажмите ctrl+c и пропишите команду:

docker-compose down
